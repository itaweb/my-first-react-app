Инструмент Create React App помогает создавать React сайт.

На компьютере должен быть установлен Node.

Переходим в директорию в которой хотим создавать наше приложение.

Находясь в нужной папке в терминале вводим команду npx create-react-app my-app

Последнее в данной команде это название проекта.

Запуск проекта - npm run start

Изображения добавляются через import как в файле App.js или 
напрямую можно разместить в папке public и давать на него ссылку.

В файле index.css даны стили для всего сайта, этот файл подключается через 
import в файле index.js

В файле App.js даны стили конкретно для этого компонента, и уже этот файл 
стилей подключается через import в файле App.js

Все функции hook начианются с use
